## Estructura del repositorio e instrucciones de ejecución

### Estructura del repositorio

Este repositorio está compuesto por una carpeta:  
- `AI_MiniMax_VCF`: En ella se encuentra el archivo ia_connect6.cpp que contiene el código del agente implementado
                    también contiene el Dockerfile que permite la ejecución del agente. Internamente contiene la carpeta pb, 
                    la cual contiene las clases generadas automáticamente por proto.


### Instrucciones de ejecución
En la carpeta raíz, misma carpeta donde se encuentra este archivo se puede:
Levantar todo: docker compose up --build
Ver el tablero: `http://localhost:8080`
Logs de la IA: docker logs -f mi-ia-1
